# Spacing

TARMAC's spacing system is built on an **8px base unit**, providing a consistent rhythm across all components and layouts. The scale includes smaller increments (2px, 4px, 6px) for fine-tuning optical alignment within components.

## Principles

- **Consistent rhythm** — The 8px grid creates visual harmony across all interfaces
- **Predictable** — Teams can make spacing decisions quickly using the defined scale
- **Responsive** — Spacing tokens adapt across breakpoints while maintaining proportions

<Info>
  Use the spacing scale tokens rather than arbitrary pixel values. This ensures consistency and makes future adjustments to the spatial system seamless.
</Info>

## Spacing Scale

| Token | Value | Pixels | Usage |
|-------|-------|--------|-------|
| `space.0` | `0` | 0px | Reset, no spacing |
| `space.2` | `0.125rem` | 2px | Micro adjustments, icon-to-text gaps |
| `space.4` | `0.25rem` | 4px | Tight internal padding, inline gaps |
| `space.6` | `0.375rem` | 6px | Small component internal spacing |
| `space.8` | `0.5rem` | 8px | Base unit — default internal padding |
| `space.12` | `0.75rem` | 12px | Compact component padding |
| `space.16` | `1rem` | 16px | Standard component padding |
| `space.20` | `1.25rem` | 20px | Comfortable component padding |
| `space.24` | `1.5rem` | 24px | Section internal spacing |
| `space.32` | `2rem` | 32px | Card padding, group spacing |
| `space.40` | `2.5rem` | 40px | Large section spacing |
| `space.48` | `3rem` | 48px | Page section gaps |
| `space.64` | `4rem` | 64px | Major layout divisions |
| `space.80` | `5rem` | 80px | Hero spacing, page-level gaps |

## Component Spacing

| Component | Padding (x / y) | Gap | Margin |
|-----------|-----------------|-----|--------|
| Button | `16px / 8px` | `8px` (icon-text) | — |
| Input | `12px / 8px` | — | `0 0 16px` |
| Card | `24px` | `16px` (content) | `0 0 16px` |
| Modal | `24px` | `16px` (content) | — |
| List Item | `16px / 12px` | `8px` | — |
| Section | `32px / 24px` | `16px` | `0 0 32px` |
| Badge | `8px / 4px` | `4px` (icon-text) | — |
| Tooltip | `12px / 8px` | — | — |

## Accessibility

<Warning>
  Ensure interactive elements (buttons, links, inputs) have a minimum touch target of 44×44px. Use spacing tokens to achieve this — for example, a small button with `space.8` vertical padding on a 16px line-height meets the 32px minimum height, but consider `space.12` for touch-friendly targets.
</Warning>

- Maintain adequate spacing between interactive elements to prevent accidental activation
- Use consistent spacing to create clear visual groupings (Gestalt proximity principle)
- Ensure spacing scales appropriately on zoom up to 200%
