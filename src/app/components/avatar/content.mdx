## Usage Guidelines

<Info>
  Always provide a `name` prop for accessibility. The component uses it for `alt` text and initials fallback.
</Info>

<Warning>
  Avoid using avatars smaller than 24px (XS) — initials become unreadable and status dots are too small to perceive.
</Warning>

---

## Accessibility

| Requirement | Guideline |
|-------------|-----------|
| Alt text | Always provide meaningful `alt` via `name` prop |
| Decorative | Use `aria-hidden="true"` if purely decorative |
| Status | Status dot needs `aria-label` (e.g., "Online") |
| Group | Avatar group needs count announcement |
| Contrast | Initials must have ≥ 4.5:1 contrast on background |
| Focus | Interactive avatars need visible focus ring |

## Developer Handoff

```tsx
interface AvatarProps {
  src?: string;
  name: string;
  size?: 'xs' | 'sm' | 'md' | 'lg' | 'xl';
  status?: 'online' | 'offline' | 'busy' | 'away';
  badge?: number;
  onClick?: () => void;
}

interface AvatarGroupProps {
  max?: number;
  size?: 'xs' | 'sm' | 'md' | 'lg' | 'xl';
  children: React.ReactNode;
}
```

```tsx
<Avatar src="/photos/user.jpg" name="Rahul Sharma" size="lg" status="online" />
<Avatar name="Priya Patel" size="md" />
<AvatarGroup max={3}>
  <Avatar name="User 1" src="/u1.jpg" />
  <Avatar name="User 2" src="/u2.jpg" />
  <Avatar name="User 3" src="/u3.jpg" />
  <Avatar name="User 4" src="/u4.jpg" />
</AvatarGroup>
```
