## Variants

| Variant | Description |
|---------|-------------|
| Default | Standard dialog with header, body, footer |
| Confirmation | Short prompt with confirm/cancel actions |
| Destructive | Red-accented for dangerous actions |
| Form | Contains form fields with submit action |
| Scrollable | Body scrolls when content overflows |
| Full-screen | Takes up entire viewport on mobile |

## Anatomy

| # | Element | Description |
|---|---------|-------------|
| 1 | Backdrop | Semi-transparent overlay behind modal |
| 2 | Container | Centered panel with shadow |
| 3 | Header | Title and optional close button |
| 4 | Body | Main content area, scrollable |
| 5 | Footer | Action buttons (primary + secondary) |
| 6 | Close Button | × icon in top-right corner |

## States

| State | Description |
|-------|-------------|
| Closed | Modal not rendered or hidden |
| Opening | Fade-in + scale animation (200ms) |
| Open | Fully visible, focus trapped inside |
| Closing | Fade-out animation (150ms) |

## Usage Guidelines

<Info>
  Use modals sparingly for critical interactions that require user attention. Provide a clear title and obvious dismiss action.
</Info>

<Warning>
  Avoid stacking multiple modals. If you need layered content, consider using a step-based flow within a single modal.
</Warning>

---

## Accessibility

| Requirement | Guideline |
|-------------|-----------|
| Role | Use `role="dialog"` with `aria-modal="true"` |
| Label | Set `aria-labelledby` pointing to title |
| Focus trap | Tab cycles within modal only |
| Focus restore | Return focus to trigger on close |
| Dismiss | Escape key closes the modal |
| Backdrop | Click outside closes (configurable) |
| Scroll lock | Prevent body scroll when open |

## Developer Handoff

```tsx
interface ModalProps {
  open: boolean;
  size?: 'sm' | 'md' | 'lg';
  closeOnBackdrop?: boolean;
  closeOnEscape?: boolean;
  onClose: () => void;
  children: React.ReactNode;
}

interface ModalHeaderProps {
  title: string;
  showClose?: boolean;
}

interface ModalFooterProps {
  children: React.ReactNode;
}
```

```tsx
<Modal open={isOpen} onClose={handleClose} size="md">
  <Modal.Header title="Confirm Cancellation" />
  <Modal.Body>
    <p>Are you sure you want to cancel this shipment?</p>
  </Modal.Body>
  <Modal.Footer>
    <Button variant="secondary" onClick={handleClose}>Go Back</Button>
    <Button variant="primary" color="error" onClick={handleConfirm}>Cancel Shipment</Button>
  </Modal.Footer>
</Modal>
```
