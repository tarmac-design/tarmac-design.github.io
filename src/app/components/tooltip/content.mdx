## Variants

| Variant | Description |
|---------|-------------|
| Default | Dark background with white text |
| Light | Light background with dark text |
| Rich | Supports headings, links, or custom content |
| With Arrow | Includes directional arrow pointing to trigger |

---

## Anatomy

| # | Element | Description |
|---|---------|-------------|
| 1 | Trigger | Element that activates the tooltip |
| 2 | Container | Floating panel with content |
| 3 | Arrow | Triangular pointer toward trigger |
| 4 | Content | Text or rich content inside |

---

## States

| State | Description |
|-------|-------------|
| Hidden | Tooltip not visible |
| Visible | Shown after hover/focus delay |
| Entering | Fade-in animation (150ms) |
| Exiting | Fade-out animation (100ms) |

---

## Usage Guidelines

<Info>
  Use tooltips for supplementary information that isn't essential. Critical information should always be visible on the page.
</Info>

<Warning>
  Never put interactive elements (links, buttons) inside a tooltip — use a Popover instead for interactive content.
</Warning>

---

## Accessibility

| Requirement | Guideline |
|-------------|-----------|
| ARIA | Use `aria-describedby` linking trigger to tooltip |
| Keyboard | Show on focus, dismiss on Escape |
| Timing | Minimum 1.5s display time, no auto-dismiss |
| Touch | Provide alternative for touch devices |
| Motion | Respect `prefers-reduced-motion` |
| Role | Use `role="tooltip"` on the container |

## Developer Handoff

```tsx
interface TooltipProps {
  content: React.ReactNode;
  placement?: 'top' | 'bottom' | 'left' | 'right';
  variant?: 'dark' | 'light';
  arrow?: boolean;
  delay?: number;
  trigger?: 'hover' | 'focus' | 'click';
  children: React.ReactElement;
}
```

```tsx
<Tooltip content="Edit shipment details">
  <IconButton icon={<EditIcon />} />
</Tooltip>
<Tooltip content="Last updated 2 hours ago" placement="bottom">
  <span>Updated recently</span>
</Tooltip>
```
