## Variants

| Variant | Style | Use Case |
|---------|-------|----------|
| Black | Primary | Main page action — "Save Changes" |
| Black | Secondary | Supporting action — "Cancel" |
| Black | Tertiary | Subtle action — "Learn More" |
| Blue | Primary | System action — "Submit", "Continue" |
| Success | Primary | Positive confirmation — "Approve" |
| Error | Primary | Destructive action — "Delete" |
| Warning | Primary | Caution action — "Override" |
| White | Primary | Dark background — "Get Started" |

## States

| State | Description |
|-------|-------------|
| Default | Resting state, ready for interaction |
| Hover | Pointer over button, slightly lighter background |
| Pressed | Active click/tap, darker background |
| Focused | Keyboard focus, 2px blue outline |
| Disabled | Non-interactive, 40% opacity |

## Accessibility

| Requirement | Guideline |
|-------------|-----------|
| Semantic HTML | Always use native `<button>` element |
| Keyboard | Tab to focus, Enter/Space to activate |
| Focus ring | 2px visible outline, ≥ 3:1 contrast |
| Text contrast | ≥ 4.5:1 against background (WCAG AA) |
| Touch target | Minimum 44×44px on mobile |
| Icon-only | Must have `aria-label` |
| Loading | Use `aria-busy="true"` |

## Developer Handoff

```tsx
interface ButtonProps {
  variant?: 'black' | 'white' | 'blue' | 'success' | 'error' | 'warning';
  style?: 'primary' | 'secondary' | 'tertiary';
  size?: 'xl' | 'lg' | 'md' | 'sm' | 'xs';
  disabled?: boolean;
  loading?: boolean;
  icon?: ReactNode;
  iconPosition?: 'left' | 'right';
  children: ReactNode;
}
```

```tsx
// Usage
<Button variant="black" style="primary" size="lg">Save</Button>
<Button variant="error" style="primary">Delete</Button>
<Button variant="blue" loading>Saving...</Button>
```
