## States

| State | Description |
|-------|-------------|
| Off | Track is neutral, thumb positioned left |
| On | Track is filled, thumb positioned right |
| Hover | Slight shadow on thumb |
| Focused | 2px blue outline around track |
| Disabled | 40% opacity, non-interactive |

## Usage Guidelines

<Info>
  Use toggles for settings that take effect immediately, like enabling notifications or dark mode.
</Info>

<Warning>
  Don't use toggles inside forms that require a submit action — use Checkboxes instead.
</Warning>

---

## Accessibility

| Requirement | Guideline |
|-------------|-----------|
| Semantic HTML | Use `<button>` with `role="switch"` |
| Keyboard | Tab to focus, Space/Enter to toggle |
| Focus ring | 2px visible outline, ≥ 3:1 contrast |
| Label | Always associate with `aria-label` or visible label |
| State | Use `aria-checked="true/false"` |
| Motion | Respect `prefers-reduced-motion` for animations |

## Developer Handoff

```tsx
interface ToggleProps {
  checked?: boolean;
  disabled?: boolean;
  size?: 'md' | 'sm';
  label?: string;
  description?: string;
  onChange?: (checked: boolean) => void;
}
```

```tsx
<Toggle label="Enable notifications" />
<Toggle checked label="Dark mode" size="sm" />
<Toggle label="Auto-sync" description="Sync data every 5 minutes" />
```
