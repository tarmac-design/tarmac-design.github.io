## Variants

| Variant | Description |
|---------|-------------|
| Default | Standard multi-line text area |
| With Label | Textarea with top-aligned label |
| With Character Count | Shows current/max character count |
| Auto-Resize | Grows vertically with content |
| Fixed Height | Set number of visible rows |
| Disabled | Non-interactive state |
| Error | Validation error state with message |

## Anatomy

| # | Element | Description |
|---|---------|-------------|

## States

| State | Description |
|-------|-------------|
| Default | Empty or filled, ready for interaction |
| Hover | Border color darkens slightly |
| Focused | 2px blue border |
| Filled | Contains user-entered content |
| Disabled | 40% opacity, non-interactive |
| Error | Red border with error message below |
| Over Limit | Character count turns red |

## Usage Guidelines

<Info>
  Use Textarea for multi-line content like comments, descriptions, or addresses. Set a reasonable `maxLength` to guide users.
</Info>

<Warning>
  Avoid setting a very small fixed height — ensure at least 3 visible rows so users can see enough context while typing.
</Warning>

---

## Accessibility

| Requirement | Guideline |
|-------------|-----------|
| Semantic HTML | Use native `<textarea>` element |
| Keyboard | Tab to focus, standard text editing keys |
| Focus ring | 2px visible outline, ≥ 3:1 contrast |
| Label | Always associate with `<label>` or `aria-label` |
| Error | Use `aria-invalid` and `aria-describedby` for errors |
| Character count | Announce limit with `aria-live="polite"` |

## Developer Handoff

```tsx
interface TextareaProps {
  label?: string;
  placeholder?: string;
  value?: string;
  rows?: number;
  maxLength?: number;
  showCount?: boolean;
  autoResize?: boolean;
  resize?: 'none' | 'vertical' | 'both';
  disabled?: boolean;
  readOnly?: boolean;
  error?: string;
  helperText?: string;
  onChange?: (value: string) => void;
}
```

```tsx
<Textarea label="Description" placeholder="Enter details..." rows={4} />
<Textarea label="Notes" maxLength={500} showCount autoResize />
<Textarea label="Address" error="Address is required" />
```
