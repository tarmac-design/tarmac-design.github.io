# TARMAC Design System Documentation

Custom documentation site for the TARMAC Design System, built with Next.js 14 + MDX + Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Pushes to `main` automatically deploy to GitHub Pages via GitHub Actions.

## Tech Stack

- Next.js 14 (App Router, static export)
- MDX for content
- Tailwind CSS for styling
- GitHub Pages for hosting
